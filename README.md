# 감성일기장 구현(for React)

1. React Router 사용
  - Path Variable(useParams)
  - Query String(useSearchParams)
    - Query: 웹페이지에 데이터를 전달하는 가장 간단한 방법
    - name 과 value를 엮어서 데이터 전송
  - Page Moving(useNavigate)
    - Link 태그를 클릭 안해도 의도적으로 페이지를 변경할 수 있다.
